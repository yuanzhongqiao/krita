<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Krita 是一款免费开源数字绘画应用程序。它适合那些想要从头到尾创作专业作品的艺术家。 Krita 被漫画艺术家、插画家、概念艺术家、遮罩和纹理画家以及数字视觉特效行业使用。</font></font></p>
